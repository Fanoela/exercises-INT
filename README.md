# INT MIT - Assignments Repository

Ce repository contient tous les devoirs INT MIT depuis mars 2026.

Il a été créé afin de regrouper tous les anciens repositories en un seul projet structuré.

Les anciens repositories ont été supprimés.

---

## Contenu

Ce repository contient :

### Exercices en C
- Affichage des nombres premiers
- Comptage du nombre de caractères dans une chaîne de caractères
- Inversion d’une chaîne de caractères donnée
- Calcul des termes d’une suite
- Manipulation vectorielle et matricielle
- Conversion d’une date en français vers une date en malgache
- Génération d’un fichier HTML à partir d’un fichier CSV

### Exercices en C++
- Programmation orientée objet
- Classes et objets
- Relations entre classes
